# React Examples

This example demonstrates how to use `form-guardian` with React and Vite.

## Examples Included

- **React Hook Form** - Integration with React Hook Form
- **Formik** - Integration with Formik
- **Plain Form** - Plain React form with `useFormAutosave` hook

## Running

```bash
pnpm install
pnpm dev
```

The example will open at `http://localhost:3001`
